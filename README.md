# Spring with JNI Examples on Linux

## How to do it ?
1. Config Gradle First  
2. Create Native using Class  
3. Create Native Library  
4. Open Gradle Tab & Make Shared Library  
5. Copy Shared Library to /usr/lib  
6. Execute it!
